Steps to Run the Project:
 1. Install Spyder
 2. Go to "open file" in "file" tab.
 3. Open "FaceMaskInit.py", "Libraries.txt", "TrainModelInit.py" files in Spyder.
 4. Install all packages from "Libraries.txt" file.
 5. After installing all Librearies, Restart Spyder Software.
 6. Run "TrainModelInit.py" file to train the model.
 7. Run "FaceMaskInit.py" file to run project to get final output.
